What it does

A software-only layer that rides on a city's existing cameras, transit feeds, and signal infrastructure — no $116K-per-intersection hardware retrofits.

🗺

Living City Graph

~420 real Seattle camera intersections, 300+ live KC Metro buses, real NWS weather, and SFD Real-Time 911 dispatches fused into one operational picture with real road geometry.

live data, no API keys
📡

Real Congestion Sensing

Every moving bus is a GPS speed probe. Per-intersection congestion is computed from real observed speeds (weighted median, freshness windows) — optionally fused with WSDOT loop-detector flow.

bus GPS × WSDOT flow
👁

AI Vision Detection

A background sweep runs Claude Haiku on live traffic-camera frames, raising real incidents through the same audited pipeline as edge telemetry — congested intersections checked first.

multimodal LLM on live frames
🤖

Safety-Gated AI Plans

Claude Sonnet proposes signal-timing mitigations — then an independent MUTCD constraint verifier, hallucination monitor, provenance check, and confidence gate decide if an operator ever sees them. The LLM is never trusted.

blocked < 2% target, proven by tests
🧑‍✈️

Human-in-the-Loop, Always

Shadow → Advisory → Live mode ladder. Every plan needs explicit operator approval; one-click rollback restores exact prior timing; auto-revert proposals fire if conditions worsen.

no autonomous execution
⛓

Tamper-Evident Audit

Every detection, recommendation, approval, and rollback lands in a hash-chained audit trail that survives restarts — incident timelines in the UI are reconstructed straight from it.

verify_chain() live in the UI

The safety moat

Each guarantee is enforced by code and proven by the automated test suite — the suite is the trust artifact.

  1. R1–R7 MUTCD guardrails — minimum greens, pedestrian clearance, yellow intervals, cycle bounds, concurrency limits, EMS-corridor protection
  2. H1–H4 hallucination monitoring — plans citing nonexistent entities or stale data are blocked before review
  3. Mandatory provenance — no sources, no weather, no rationale → suppressed
  4. Confidence abstention — below threshold, the AI says "insufficient data confidence" instead of guessing
  5. Mode ladder — no code path executes a physical change in Shadow or Advisory mode
  6. Privacy at the edge — unredacted telemetry is rejected at ingestion; raw video never enters the platform

Quick start

Python 3.10+ and nothing else. The whole platform — server, AI gate, simulation, UI — is standard library.

quick start
# run the test suite (network-free)
python -m unittest discover -s platform/tests -t platform

# launch with live Seattle data (default), or:
python platform/run.py --city tacoma   # second city, zero new keys
python platform/run.py --sim           # fully offline demo mode

# or containerized
docker compose up --build
